# Pommes de Terre

An e-commerce web application of the world's most luxurious boutique offering exotic and unique potatoes.

# Deployed on

https://pommes-de-terre.herokuapp.com/

# Features

Two Shopping Experiences: Guest and User
View all products
Cart
Checkout
Order History
User Profile
Stripe Integration

# Resources
